About AlienXFile V2

AlienXFile V2 is a secure, fast, and temporary file/text sharing platform built with Python and Flask. Users can upload files or text, receive a unique 4-digit key, and download content via the Catbox.moe API. All uploads are temporary and automatically removed, ensuring privacy and security.

Easy Upload

Upload files up to 200MB or share text with a simple interface

Secure Access

Automatic 4-digit key generation for secure content access

Temporary Storage

Files expire automatically, ensuring your privacy and security

Modern UI

Responsive design with smooth animations for great user experience
